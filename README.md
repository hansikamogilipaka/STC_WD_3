# 🧠 Quiz Game Application

An interactive and user-friendly quiz game web application developed using HTML, CSS, and JavaScript. This project allows users to answer multiple-choice questions, track their progress, and view their final score at the end.

---

## 🚀 Features

- 📋 Multiple-choice questions
- 🎯 Score tracking system
- ✅ Answer validation with correct/wrong highlighting
- 📊 Question progress indicator
- 🔄 Restart quiz functionality
- 🎨 Modern and responsive UI design

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
